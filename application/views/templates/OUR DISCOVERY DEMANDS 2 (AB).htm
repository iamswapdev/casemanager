-------------------------------------------------------------------X

 

 

 

PLEASE TAKE NOTICE plaintiff hereby demands, pursuant to CPLR 3130, 3131, and 3133, defendant serve upon plaintiff, within twenty (20) days, answers to each of the following interrogatories:

INTERROGATORIES

INTERROGATORY 1- Concerning Each Cause of Action:

  1. Set forth the full name, title, address and relationship to defendant of the person answering these interrogatories.

INTERROGATORY 2- Concerning Each Cause of Action:

  1. Set forth if at the time of the motor vehicle accident there existed a valid insurance policy issued by defendant providing No-Fault benefits for the assignor(s) herein.
  2. Set forth a true and accurate copy of any such policy.

INTERROGATORY 3- Concerning Each Cause of Action:

  1. Set forth if defendant received an NF-2 concerning the assignor(s) herein.
  2. Set forth when the NF-2 was received by defendant.
  3. Set forth a true and accurate copy of the NF-2.

INTERROGATORY 4- Concerning Each Cause of Action:

  1. Set forth if defendant received an MV-104 concerning the assignor(s) herein.
  2. Set forth a true and accurate copy of the MV-104.

INTERROGATORY 5- Concerning Each Cause of Action:

  1. Set forth if defendant received a Police Accident Report.
  2. Set forth a true and accurate copy of the Report.

INTERROGATORY 6- Concerning Each Cause of Action:

  1. Set forth if defendant received the bill(s) that are the subject of this action.
  2. Set forth the date said bill(s) were received by defendant.
  3. Set forth a true and accurate copy of the bill(s).

INTERROGATORY 7- Concerning Each Cause of Action:

  1. Set forth if defendant received an assignment of benefits form for the bill(s) that are the subject of this action.
  2. Set forth the date the assignment of benefits form was received.
  3. Set forth a true and accurate copy of the assignment of benefits form.
  4. Set forth if defendant denied plaintiff's bill(s), in whole or in part, based on the failure to provide an assignment of benefits.
  5. Set forth if defendant requested any additional verification concerning an assignment of benefits.

INTERROGATORY 8- Concerning Each Cause of Action:

  1. Set forth if defendant made any requests for additional verification for the bill(s) that are the subject of this action.
  2. Set forth what additional verification was requested.
  3. Set forth the reason why the requested verification was necessary to process the claim.
  4. Set forth the date such request(s) were mailed.
  5. Set forth to whom the request(s) were mailed.
  6. Set forth a true and accurate copy of the requests.
  7. Set forth if defendant received any response to the requests.
  8. Set forth the date the response was received.
  9. Set forth a true and accurate copy of the response.

INTERROGATORY 9- Concerning Each Cause of Action:

  1. Set forth if defendant made any payment for the bill(s) that are the subject of this action.
  2. Set forth the amounts of the payments made.
  3. Set forth the date the payments were made.
  4. Set forth a true and accurate copy of the front and back of the drafts issued.

INTERROGATORY 10- Concerning Each Cause of Action:

  1. Set forth if defendant denied, in whole or in part, for the bill(s) that are the subject of this action.
  2. Set forth the reason plaintiff's bill(s) was denied.
  3. Set forth the date the denial(s) was mailed.
  4. Set forth to whom the denial(s) was mailed.
  5. Set forth a true and accurate copy of the denial(s).

INTERROGATORY 11- Concerning Each Cause of Action:

  1. If the claim was denied based upon a peer review or Independent Medical Examination (IME) set forth a copy of the peer or IME report.
  2. Set forth true and accurate copies of all documents forwarded to the reviewer concerning the peer review or IME.
  3. Set forth true and accurate copies of all medical records maintained by defendant in the assignor's No-Fault file not forwarded to the reviewer.

INTERROGATORY 12- Concerning Each Cause of Action:

  1. Set forth if defendant alleges the assignor was not injured or that the assignor's injuries were not causally related to the motor vehicle accident.
  2. Set forth in detail how the defendant came to this conclusion.
  3. Set forth any medical reports or other documents relied upon by defendant in coming to this conclusion.
  4. Set forth any biomechanical or any other report relied upon by defendant in coming to this conclusion.

INTERROGATORY 13- Concerning Each Cause of Action:

  1. Set forth if defendant alleges plaintiff's bill is in violation of the applicable fee schedule.
  2. Set forth any ground rule or computations relied upon by defendant and how such are applicable to plaintiff's bill(s).

INTERROGATORY 14- Concerning Each Cause of Action:

  1. Set forth if defendant alleges the underlying motor vehicle accident was an intentional act.
  2. Set forth if defendant assigned plaintiff's claim to a Special Investigation Unit(SIU).
  3. Set forth the name of the investigator assigned.
  4. Set forth the name of any database(s) utilized by defendant in conducting its investigation and the results of any searches of such database(s).
  5. Set forth if defendant, in conducting its investigation, attempted to obtain, or did obtain, any statements, written or oral, of any person. If so, provide true and accurate copies of any such statements or transcripts and details of attempts made to obtain such statements.
  6. Set forth if defendant, in conducting its investigation, attempted to obtain, or did obtain, any examinations under oath. If so, provide true and accurate copies of any notices or transcripts concerning such.
  7. Set forth if defendant, in conducting its investigation, prepared any SIU report. If so, provide true and accurate copies of any such reports.
  8. Set forth if defendant, in conducting its investigation, obtained any police accident reports. If so, provide true and accurate copies of any reports.
  9. Set forth if defendant, in conducting its investigation, obtained any photographs of vehicles, persons, accident scenes, buildings or other locales. If so, provide true and accurate copies of such photographs.
  10. Set forth if defendant made any filing with the Insurance Frauds Bureau. If so, provide true and accurate copies of any filings.
  11. Set forth true and accurate copies of all notes, memoranda, reports or other documents relied upon by defendant in coming to the conclusion the underlying motor vehicle accident was an intentional act.
  12. Set forth true and accurate copies of all notes, memoranda, reports or other documents intended to be introduced into evidence by defendant in this matter concerning the allegation that the underlying motor vehicle accident was an intentional act.

PLEASE TAKE NOTICE, that this demand is a continuing demand, and in the event knowledge is acquired subsequent to the receipt of this demand, up to and including the trial of this action, you are required to set forth in writing, and under oath, the name and address of such persons.

PLEASE TAKE NOTICE, in the event this Demand is not complied with, or if any part of this Demand is objected to, plaintiff will, upon the trial of this action, move the Court to preclude and forbid the testimony of any witness offered by the defendant as to what has not herein been furnished to the plaintiffs attorneys, pursuant to this Demand.
